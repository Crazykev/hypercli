Hyper_ client for Mac/Linux

How to build
```
$ mkdir $GOPATH/hyperhq/
$ cd $GOPATH/hyperhq/
$ git clone https://github.com/hyperhq/hypercli hypercli
$ cd hypercli
$ ./build-hyperserve-client.sh
```
